Salon Management System (DESIGN & FRONTEND)

WEB DEVELOPER (DJANGO/PYTHON)

I developed a Salon Management System using Django, offering a full-featured web application to manage salon operations. The system includes secure user authentication with OTP-based password reset, interactive dashboards with Chart.js for sales and performance analytics, and complete CRUD functionality for workers, products, and services. It features a robust cart and checkout system with tax, discount, and tip calculations, along with PDF and 80mm thermal bill printing. RESTful APIs support client management with search and pagination, while advanced reporting and real-time updates enhance usability. Built with Python, Django, PostgreSQL, AJAX, and xhtml2pdf, the system ensures efficient data handling, error management, and secure session control.
